# Gestion Hôtel - Application de Gestion Hôtelière

##  Description
Application développée avec Python et CustomTkinter pour la gestion d'un hôtel (chambres, clients, réservations, factures).

---

## Prérequis

Avant d'installer l'application, assurez-vous d'avoir :

 Logiciel ---> Version requise 
 Python  ---> 3.8 ou supérieur 
 MySQL   ---> 5.7 ou supérieur 
 pip     ---> Dernière version

---

##  Installation

### Étape 1 : Installer Python
- Téléchargez Python sur [python.org](https://www.python.org/downloads/)
- **Cochez** "Add Python to PATH" pendant l'installation

### Étape 2 : Installer MySQL
- Téléchargez MySQL sur [mysql.com](https://dev.mysql.com/downloads/installer/)
- Installez MySQL Server
- Notez le mot de passe root que vous avez choisi

### Étape 3 : Télécharger le projet
- Placez le dossier `GestionHotel` sur votre bureau ou dans vos documents

### Étape 4 : Installer les bibliothèques Python
Ouvrez un terminal (cmd) dans le dossier du projet et exécutez :

pip install customtkinter
pip install mysql-connector-python

## Configuration de la base de données
### Étape 1 : Ouvrir MySQL

mysql -u root -p
Entrez votre mot de passe root.

### Étape 2 : Créer la base de données

CREATE DATABASE hotel_gestion;
USE hotel_gestion;
### Étape 3 : Créer les tables
Exécutez ces commandes SQL dans l'ordre :


CREATE TABLE admins (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    email VARCHAR(100)
);

CREATE TABLE chambres (
    id INT PRIMARY KEY AUTO_INCREMENT,
    numero VARCHAR(10) UNIQUE NOT NULL,
    type ENUM('simple', 'double', 'suite') NOT NULL,
    prix_nuit DECIMAL(10,2) NOT NULL,
    statut ENUM('libre', 'occupee', 'nettoyage') DEFAULT 'libre'
);

CREATE TABLE clients (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nom VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    telephone VARCHAR(20),
    adresse TEXT
);

CREATE TABLE reservations (
    id INT PRIMARY KEY AUTO_INCREMENT,
    client_id INT NOT NULL,
    chambre_id INT NOT NULL,
    date_arrivee DATE NOT NULL,
    date_depart DATE NOT NULL,
    statut ENUM('active', 'terminee', 'annulee') DEFAULT 'active',
    FOREIGN KEY (client_id) REFERENCES clients(id) ON DELETE CASCADE,
    FOREIGN KEY (chambre_id) REFERENCES chambres(id) ON DELETE CASCADE
);

CREATE TABLE factures (
    id INT PRIMARY KEY AUTO_INCREMENT,
    reservation_id INT NOT NULL,
    montant_total DECIMAL(10,2) NOT NULL,
    date_paiement DATE,
    mode_paiement ENUM('carte bancaire', 'espèces', 'virement', 'chèque'),
    FOREIGN KEY (reservation_id) REFERENCES reservations(id) ON DELETE CASCADE
);
